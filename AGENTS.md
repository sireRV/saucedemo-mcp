## Test Execution & Workflow Commands

Always use the following commands inside the environment via your execution tools. Do not guess or deviate.

- **Run all tests:** `npx playwright test`
- **Run a specific test file:** `npx playwright test tests/example.spec.ts`
- **Run in headed mode:** `npx playwright test --headed`
- **Debug tests:** `npx playwright test --debug`
- **Prevent report auto-opening:** Set the environment variable `PW_TEST_HTML_REPORT_OPEN=never` during execution.
